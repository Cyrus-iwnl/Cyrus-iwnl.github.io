本文记录了我今天学习的内容，主要包括包、变量、函数、控制结构、指针、数组、切片、map等基本语法知识。

包、变量与函数

包（Package）

Go 程序由包组成。每个 Go 源文件都需要以 package 声明开头。常见的 main 包表示可独立运行的程序，其入口为 main() 函数。

导入（Import）

通过 import 关键字导入其他包，例如：

1

import "fmt"

导出名（Exported Name）

在 Go 中，以大写字母开头的标识符是可导出的（即公有），小写开头的则是私有的。

函数（Function）

函数使用 func 关键字定义，例如：

1
2
3

func add(a int, b int) int {
    return a + b
}

多返回值

Go 支持一个函数返回多个值：

1
2
3

func swap(x, y string) (string, string) {
    return y, x
}

带名字的返回值

返回值可以具名：

1
2
3
4
5

func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return
}

变量

声明变量

使用 var 声明变量：

1

var i int

初始化变量

可以同时声明并初始化：

1

var j int = 1

短变量声明

函数内部可以使用更简洁的 := 方式：

1

k := 3

基本类型

Go 提供诸如 int, float64, bool, string 等基本类型。

零值

未显式初始化的变量会被赋予“零值”。

类型转换与类型推断

Go 不支持隐式类型转换，必须显式转换：

1
2

var i int = 42
var f float64 = float64(i)

Go 具有类型推断能力：

1

v := 42 // 推断为 int 类型

常量

使用 const 定义常量：

1
2


const Pi = 3.14

控制结构

for 循环

Go 仅有一种循环结构：for。

1
2
3

for i := 0; i < 10; i++ {
    fmt.Println(i)
}

无限循环

1
2
3

for {
    // ...
}

for 是 Go 中的 while

可用 for 模拟 while：

1
2
3

for condition {
    // ...
}

求值顺序

Go 的 switch 自动在 case 匹配成功后退出，无需 break。

无条件 switch

1
2
3
4
5
6
7
8

switch {
case x > 0:
    fmt.Println("Positive")
case x < 0:
    fmt.Println("Negative")
default:
    fmt.Println("Zero")
}

defer 延迟调用

defer 会在函数返回前调用指定的语句，常用于资源释放。

1
2
3
4

func main() {
    defer fmt.Println("world")
    fmt.Println("hello")
}

调用顺序为后进先出（LIFO）
